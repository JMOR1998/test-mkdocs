Codigo externo al HTML

Existen otros lenguajes de programación a demás de HTML, como por ejemplo CSS o Python.

CSS

¿Qué es?

Es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita. Funciona junto con el lenguaje HTML que se encarga del contenido básico de las páginas.

¿Cómo se usa?

Dentro del repositorio donde tenemos nuestra página HTML añadimos un fichero de texto con extensión .css y en su interior añadimos el siguiente contenido:
p { color: red; }

Python

¿Qué es?

Python es un lenguaje de programación de alto nivel que se utiliza para desarrollar aplicaciones de todo tipo. A diferencia de otros lenguajes como Java o .NET, se trata de un lenguaje interpretado, es decir, que no es necesario compilarlo para ejecutar las aplicaciones escritas en Python, sino que se ejecutan directamente por el ordenador utilizando un programa denominado interpretador, por lo que no es necesario “traducirlo” a lenguaje máquina.

Ejemplo de código python3

a = 2000
b = 21
print(f"a = {a}")
print(f"b = {b}")
print(f"a + b = {a + b}")
